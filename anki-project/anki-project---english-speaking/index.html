      Anki Project - English Speaking

      2025년 7월 2일

      개요

      Anki를 통해 이제까지 단어/숙어를 암기해 왔다. 이번에는 좀 더 직접적으로 아웃풋 과정에 도움이 될 만한 학습 방식을 설계하여 직접 실험해 보고자 한다.

      덱 설계

      기본 전략은 어학 말하기&쓰기 시험의 기본 유형에서 논의한 대로 유형을 분류하고, 각 유형에 대해 적절한 지문을 골라 Chunk cloze 방식을 사용해 Anki로 암기하는 것이다.

      보이는 대로 표현하기 (Picture Description)
      • 적절한 사진을 고른다. DET Practice나 Gurully 등의 사이트에서 DET 이미지 묘사 유형의 이미지와 샘플 답안을 제공하는데, 이 샘플 답안보다는 Gemini가 제공하는 답안이 더 구조적 일관성을 가져 학습에 적합하다. 여기서 이미지만을 추출해서 Gemini에게 답안 작성 및 Ankify를 지시할 것이다.
      • 사용 프롬프트
      당신은 듀오링고 영어 시험(Duolingo English Test)이나 토익 스피킹(TOEIC Speaking)과 같은 영어 말하기 시험을 전문적으로 가르치는 영어 튜터입니다. 당신의 목표는 제가 제공하는 이미지를 기반으로, 저의 이미지 묘사 실력을 향상시킬 수 있는 고품질의 학습 자료를 만드는 것입니다.

제가 이미지를 제공하면, 당신은 반드시 아래의 두 가지 작업을 순서대로 수행해야 합니다.

Task 1: 상세 영어 묘사 지문 생성

제공된 이미지를 상세하고, 풍부하며, 자연스러운 영어 표현을 사용해 묘사합니다.

묘사의 구조는 다음 논리적 순서를 따릅니다:

전반적인 첫인상 및 상황 요약

중심이 되는 인물 또는 사물에 대한 상세 묘사

주변 환경 및 배경 묘사

이미지로부터의 추론, 분위기, 느낌 묘사

전체 묘사 지문을 하나의 텍스트 블록으로 작성합니다.

결과물은 복사하기 쉽도록, 반드시 세 개의 백틱(```)으로 시작하고 끝나는 코드 블록 형식으로 제공해 주세요.

코드 블록 바로 위에는 ### Detailed Image Description 라는 제목을 붙여주세요.

Task 2: Anki Cloze 버전 생성

Task 1에서 당신이 생성한 정확히 동일한 묘사 지문을 사용합니다.

각 문장을 의미 있는 단위(청크)로 나눕니다.

각 청크를 {{c#::영어 청크::한국어 번역}} 형식의 Anki Cloze로 변환합니다.

예시 (Example):

원문: The photo shows a young woman. She is sitting at a café.

변환 결과: {{c1::The photo shows::사진은 보여줍니다}} {{c2::a young woman.::한 젊은 여성을.}} {{c3::She is sitting::그녀는 앉아 있습니다}} {{c4::at a café.::카페에서.}}

매우 중요: Cloze 카운터(c#)는 리셋 없이 문단 전체에 걸쳐 계속 이어져야 합니다 (c1, c2, c3, ... cN). 이렇게 하면 전체 문단이 Anki 카드 한 개에 모두 포함됩니다.

결과물 전체를 복사하기 쉽도록, 반드시 세 개의 백틱(```)으로 시작하고 끝나는 코드 블록 형식으로 제공해 주세요.

코드 블록 바로 위에는 ### Anki Cloze Version (Copy-Paste Ready) 라는 제목을 붙여주세요.
      [예정] 일상 생활 표현하기 (Daily Life)
      • OPIC의 샘플 답안/표현을 Chunk cloze 방식으로 암기한다.
      • 김태운의 교재를 참고한다.
      • 위의 프롬프트에서 Task 2만을 약간 수정해 사용하면 될 것 같다.
      [예정] 논리적으로 표현하기 (Argumentative Expression)
      • TOEFL Speaking의 샘플 답안/표현을 Chunk cloze 방식으로 암기한다.
      • 피유진의 교재를 참고한다.

      얼마나 학습해야 할까?

      • 이미지 묘사 유형의 경우, 한 이미지당 약 35.2개의 청크가 사용된다. 만약 100개 이미지를 공부한다면 3520개 정도를 공부해야 한다.
