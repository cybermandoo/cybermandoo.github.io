      프롬프트

      
작업 목표

제공된 텍스트(강의 노트 OCR 결과)에서 모든 'Definition' 항목을 추출하여 Anki 카드에 바로 붙여넣을 수 있는 순수 텍스트 형식으로 변환한다.

처리 규칙

추출 대상:

'Definition'이라고 명시된 모든 내용을 빠짐없이 추출한다.

추출된 내용은 원문(영어)을 그대로 유지한다.

내용 정제:

페이지 번호(### Page n), 'Definition'과 같은 레이블, 목록 기호(*) 등 텍스트 내용과 직접 관련 없는 모든 메타데이터를 제거한다.

굵은 글씨(**), 백틱(`) 등 모든 마크다운 서식을 제거한다.

수식 변환 (LaTeX):

텍스트에 포함된 모든 수학적/논리적 표기는 LaTeX 형식으로 변환한다.

모든 LaTeX 수식은 Anki에서 인식할 수 있도록 \(와 \)로 감싼다.

세부 변환 규칙:

단일 기호: R, x, xRy와 같이 단독으로 쓰이는 변수나 관계도 모두 LaTeX로 변환한다. 예: \(R\), \(x\), \(xRy\).

논리 연산자: 영어 단어로 된 논리 연산자는 아래의 LaTeX 기호로 대체한다.

and → \land

or → \lor

for all → \forall

implies → \implies

iff → \Leftrightarrow

논리 구조 분리:

수량자 분리 (신규 규칙): \forall로 시작하는 수량자 부분은 그 뒤에 오는 조건/결론 부분과 별개의 LaTeX 블록으로 분리한다. 쉼표(,)는 블록 밖에 둔다.

조건/결론/좌우변 분리: \Leftrightarrow 또는 \implies 또는 = 기호는 좌변과 우변의 LaTeX 블록과 분리하여 별개의 블록으로 처리한다.

적용 예시:

원본 문장: symmetric if, for all x, y in X, xRy implies yRx.

최종 결과: symmetric if, \(\forall x, y \in X\), \(xRy\) \(\implies\) \(yRx\).

최종 결과물:

위의 모든 규칙을 적용한 최종 결과물은 다른 서식 없이, 복사-붙여넣기가 용이하도록 단일 텍스트 스니펫(코드 블록)으로 제공한다.
