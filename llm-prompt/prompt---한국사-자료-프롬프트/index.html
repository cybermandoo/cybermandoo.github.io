      한국사 자료 프롬프트

      2025년 8월 26일

      개요

      • 최태성 선생님의 한국사능력검정시험 공개 자료인 빈칸채우기 자료를 Anki에 걸맞게 변형하는 과정을 보여주고 있다.
      • 사실 아래 3개의 프롬프트를 합칠 수 있기도 하지만, 자료 양이 적지는 않아서 여러 단계에 걸쳐서 하지 않으면 AI의 출력 품질을 보장할 수 없어 나눠서 작업했다.
      • 2번 프롬프트는 AI에게 프롬프트를 만들어달라고 해서 얻은 것이다.

      1. 기존 자료를 Anki Cloze로 변환

      이 PDF는 한국사 교재의 빈칸채우기 연습 자료야. 기본적으로 빈칸이 뚫려 있고, 짝수 번째 페이지(지면에 표시되는 쪽수 말고 파일 자체의 쪽수)에는 빈칸의 답안이 적혀 있는 것을 확인할 수 있을 거야. 2강부터 40강까지 아래의 작업을 전부 수행한다.
      1단계. 네가 빈칸이 채워진 완전한 문장을 만든 다음,
      2단계. 원래 빈칸이었던 키워드의 경우는 anki cloze 처리해서 내게 보여 줘. 이 때, 여러 개의 빈칸이 한 문장에 있는 경우, 그 cloze가 동시에 표시되도록 해서는 안 되고(여러 cloze가 같은 번호여서는 안 되고), 별도로 cloze가 표시되도록 해야 해(다른 번호 부여).

      강의 번호, 강의 제목, 문장 번호, 문장 이 순서대로, tsv 형식으로 작성한다. 처음부터 끝까지 작성해 줘.

      2. Cloze化한 자료를 다시 개선

      You are an expert in Korean history and a specialist in creating effective Anki flashcards.

      Your task is to correct and optimize a list of Anki cloze notes. The provided text will already contain poorly formatted cloze deletions (e.g., {{c1::...}}). Your primary job is to completely disregard the existing clozes and regenerate a new, perfectly formatted cloze string based on the five cardinal rules below.

      Here are the five cardinal rules you must follow:

      Rule 1: Identify Core Keywords Only Identify all historically significant “core keywords” in the sentence, such as people, events, places, documents, laws, systems, organizations, treaties, and key dates/periods.

      Rule 2: Cloze the Entire Conceptual Unit The cloze must encompass the entire meaningful keyword. Do not split a concept unnecessarily.

      • Correct: {{c1::쌍성총관부}}
      • Incorrect: {{c1::쌍성}}총관부

      Rule 3: Ensure Sufficient Context (The Clue Rule) The cloze must be surrounded by enough context for the user to understand what type of information is being asked.

      • For Dates: Leave the units (년, 월, 일) outside the cloze.
        • Correct: {{c1::1920}}년 {{c2::10}}월
        • Incorrect: {{c1::1920년 10월}}
      • For Conceptual Terms: Leave words indicating the type of concept (e.g., 제도, 조약, 사업, 운동) outside the cloze when it clarifies the question.
        • Correct: {{c1::화폐 정리}} 사업
        • Incorrect: {{c1::화폐 정리 사업}}

      Rule 4: Use Sequentially Numbered Clozes If a sentence contains multiple keywords to be tested, create a single output line assigning each a unique, sequential cloze number (e.g., {{c1::keyword1}}, {{c2::keyword2}}…).

      Rule 5: Avoid Trivial Clozes Never cloze a single, meaningless syllable. The clozed portion must be a meaningful word or concept on its own.

      • Correct: {{c1::단발령}}
      • Incorrect: {{c1::단}}발령

      Execution Example:

      Input (with original, incorrect clozes):

      1920년 10월, 김좌진이 이끈 북로 군정서와 독립군 연합 부대가 {{c1::청산}}리 일대에서 일본군에게 대승을 거두었다.
박정희 정부의 7차 개헌으로 이른바 {{c1::유신}} 헌법이 만들어졌으며, 이에 따라 통일 주체 국민 회의에서 대통령을 {{c2::간}}선제로 선출하였다.
옥저에는 {{c1::민며느리}}제라는 혼인 풍습이 있었다.
{{c1::화폐}} 정리 사업은 백동화를 제일 은행권으로 바꾸는 사업이었으나, 등가 교환을 해주지 않아 국내 은행, 상인 자본가들의 화폐 자산이 감소하는 결과를 가져왔다.

      Correct Output (ignoring the old clozes and creating new ones):

      {{c1::1920}}년 {{c2::10}}월, {{c3::김좌진}}이 이끈 {{c4::북로 군정서}}와 독립군 연합 부대가 {{c5::청산리 전투}}에서 일본군에게 대승을 거두었다.
{{c1::박정희 정부}}의 {{c2::7차}} 개헌으로 이른바 {{c3::유신}} 헌법이 만들어졌으며, 이에 따라 {{c4::통일 주체 국민 회의}}에서 대통령을 {{c5::간선}}제로 선출하였다.
{{c1::옥저}}에는 {{c2::민며느리제}}라는 혼인 풍습이 있었다.
{{c1::화폐 정리}} 사업은 백동화를 제일 은행권으로 바꾸는 사업이었으나, 등가 교환을 해주지 않아 국내 은행, 상인 자본가들의 화폐 자산이 감소하는 결과를 가져왔다.
